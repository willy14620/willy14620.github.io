C 語言入門

發表於 2020-07-22 | 分類於 程式語言 ， C/C++ |

開始之前

PS：內容適用於初學者，若有程式基礎的可跳過！

先到Dev C++下載編譯器，並安裝。

程式碼基本架構

變數

  • 整數型態 (int, long, long long)
  • 浮點數型態 (float, double, long double)
  • 字元型態 (char)PS:在C裡頭字串是字元陣列

陣列

  • 差異
    變數:取一個空間存放一個變數。
    陣列:取一個連續空間放很多個整數。
  • 用法
    • 整數型態
    • 浮點數
    • 字元

PS：宣告陣列時請記得要告知長度為何，否則要使用動態配置記憶體(malloc()函數)。

printf/scanf 印出/儲存

  • 於螢幕上印出程式結果 (printf)
  • 將輸入儲存到變數中 (scanf)

if/else 條件判斷

  • 條件判斷，顧名思義當條件為真時，才進行if/else內的程式碼；反之跳過。
  • 巢狀式

switch/case 條件判斷

  • 與if/else相同功能，視情況做使用。
  • 用法

for/while 迴圈

  • 迴圈目的:執行相同程式碼多次
  • for

PS：遞增 { 變數++ } 、 遞減 { 變數– }

  • while
  • do while

PS：使用while或do-while時，要注意條件判斷，否則可能會造成無窮迴圈。

指標

PS：偏難，建議先熟練基礎語法再學。

待補

其他

如果覺得自己已經熟練語法，可以試試去UVA挑戰題目。
